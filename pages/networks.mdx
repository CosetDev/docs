# Supported Networks

Currently, only testnet networks are supported on Coset. You can still switch to mainnet networks on Coset web app but since there are no deployed contracts, you won't be able to deploy oracles.

| Network                  |   Chain ID   |
|--------------------------|--------------|
| Mantle Sepolia Testnet   |     5003     |
| Mantle                   |     5000     |
| Cronos Testnet           |     338      |
| Cronos                   |      25      |
